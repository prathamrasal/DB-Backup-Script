# DB-Backup-Script

To run the script
Step 1:
```bash
npm init
```
Step 2:
```bash
npm i 
```

# if you want to copy one database to another then
Step 3:
```bash
node copyDatabase.js
```
# if you want to generate Excel of database with multiple sheets as colletion then
Step 3: 
```bash
node generateExcel.js
```
